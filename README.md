### 2. Agentic AI Capstone Project ‚Äî F.A.I.R. (Financial Analysis & Integrated Risk)

**Repository:**  
üëâ https://github.com/shre-yes/AgenticAICapstoneProject

**Overview:**  
F.A.I.R. is a multi-agent AI system designed to deliver comprehensive, risk-adjusted
financial advice by simulating the workflow of a human financial advisory team.
The system integrates technical analysis, fundamental analysis, risk management,
and behavioral evaluation into a structured decision-making pipeline.

**Key Capabilities:**
- Multi-agent collaboration with specialized analytical roles
- Structured advisory workflow: Data ‚Üí Strategy ‚Üí Execution ‚Üí Risk ‚Üí Behavioral
- Dedicated risk management enforcing position sizing and stop-loss logic
- Behavioral analysis to flag market mania (fear/greed) and cognitive biases
- Secure handling of sensitive configuration using environment variables

**Tech Stack:**
- Language: Python
- LLM Provider: Gemini
- Agent Framework: Google ADK
- Architecture: Role-based multi-agent advisory system
- Configuration: Environment variables for secure key management
- Execution: Local CLI / Web UI

**Why it matters:**  
This project demonstrates real-world **agentic AI system design** by combining
autonomy, risk awareness, and behavioral intelligence. It showcases how AI agents
can collaborate in a disciplined, explainable workflow to support high-stakes
financial decision-making.
